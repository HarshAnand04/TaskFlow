# 🚀 TaskFlow - Task Management Web App

## 📌 Overview
TaskFlow is a full-stack task management web application designed to help users efficiently organize and manage their daily tasks. It allows users to create, update, prioritize, and delete tasks with deadlines, improving productivity and workflow management.

---

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB (Mongoose ODM)  

---

## ✨ Features
- 📝 Create, update, and delete tasks  
- 📊 Task status tracking (To-Do / In Progress / Done)  
- 🔥 Priority tagging for better organization  
- ⏰ Deadline management with due-date highlighting  
- 🔍 Filter tasks based on status  
- ⚡ Real-time UI updates without page reloads  
- 📱 Responsive and clean user interface

🌐 API Endpoints
GET /tasks → Fetch all tasks
POST /tasks → Create a new task
PUT /tasks/:id → Update an existing task
DELETE /tasks/:id → Delete a task
